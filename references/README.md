# KOGO downstream reference set

이 폴더는 최휘수 파트(SVG, spatial domain clustering, CCI, SPIX) 이론 강의자료를
업데이트할 때 바로 참고할 논문 목록입니다.

`pdf_bundle/`에는 공식 publisher PDF 또는 공개적으로 접근 가능한 PDF만 넣었습니다.
SpatialDE처럼 공식 PDF URL이 현재 HTML로 반환되는 항목은 PDF를 넣지 않고 DOI 링크만
manifest에 남겼습니다.

## 파일

- `kogo_spatial_downstream_references.tsv`: 슬라이드/문헌정리용 표
- `pdf_bundle/`: 공유용 PDF 묶음
- `pdf_bundle/README.md`: PDF 파일명과 checksum

## 파트별로 먼저 볼 논문

- SVG: SpatialDE, SPARK, nnSVG, Squidpy
- Spatial domain: BayesSpace, SpaGCN, BANKSY
- CCI: CellPhoneDB, CellChat, LIANA+, COMMOT, SpaTalk, CCI method comparison
- SPIX: 내부 SPIX manuscript와 이 repo의 Colab validation 결과를 같이 사용
